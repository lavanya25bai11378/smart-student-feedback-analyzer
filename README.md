# Smart Student Feedback Sentiment Analyzer

## Description
This project automatically analyzes student feedback and classifies it as Positive, Neutral, or Negative. It helps educators quickly understand students’ opinions, identify areas of improvement, and save time in reviewing feedback manually. The tool uses Machine Learning, Natural Language Processing (NLP), and Probability concepts from AI to provide accurate and visual insights.

## Features
- Classifies feedback as Positive, Neutral, or Negative
- Visualizes overall sentiment using a pie chart
- Shows top keywords in positive and negative feedback with bar charts or word clouds
- Accepts new feedback input and predicts sentiment instantly
- Fully automated and easy to use

## Course Concepts Applied
- **Machine Learning (ML):** Supervised classification using Naive Bayes
- **Probability for ML:** Understanding predictions and class probabilities
- **NLP & Feature Learning:** Text preprocessing and TF-IDF conversion
- **Intelligent Agents:** Automatic analysis and recommendation based on input
- **Visualization:** Clear visual outputs for decision-making
